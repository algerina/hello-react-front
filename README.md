## Hello React front
This is a React front-end app that displays a random greeting message.

## Setup
Clone this repository
$ git clone git@github.com:Sahar-AbdelSamad/hello-react-front-end.git
$ cd hello-react-front-end
## Run project
$ npm install
$ npm run start
